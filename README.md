📚 Tableros Interactivos — Colección de Proyectos Web (realizados en Tableau)

Este repositorio reúne tres proyectos web independientes, desarrollados con fines educativos y demostrativos. Cada uno aborda una temática distinta y utiliza tecnologías modernas para la construcción de interfaces dinámicas, visualizaciones y experiencias interactivas.

📁 Contenido del repositorio
El repositorio incluye las siguientes carpetas principales:

Cv - Dinámico/  
Proyecto que presenta un currículum interactivo, con secciones dinámicas y navegación fluida.

Fútbol Argentino/  
Proyecto orientado al análisis o visualización de datos relacionados con el fútbol argentino durante las temporadas 2008 - 2022.

Tablero de Control/  
Aplicación React + Vite que implementa un tablero interactivo de auditoría presupuestaria, correspondiente al ejercicio fiscal 2012 de la Secretaría de Control Ecológica. Incluye visualización de ejecución por fuente de financiamiento y concepto de gasto, con observaciones y recomendaciones.
